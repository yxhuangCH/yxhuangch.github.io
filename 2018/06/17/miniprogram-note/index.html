2. Flex 布局

主要思想：通过设定容器（flex container 即伸缩容器）与子元素（flex item 即伸缩项目）的规则，使所有 view 组件在主轴（main axis） 与侧轴(cross axis) 上合理地自动分配。
详细可以参考阮一峰的博客 Flex 布局教程
